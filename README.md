# 📊 PakStats 

**PakStats** is an interactive data visualization platform that translates complex national data into understandable insights. Built with **React** and **Vite**, the application fetches real-time statistics from the **World Bank Open Data API** to present a clear picture of Pakistan’s development across multiple sectors.

🔗 **Live Demo:** [pak-stats.vercel.app](https://pak-stats.vercel.app)

---

## ✨ Features

* **Real-Time Data:** Integrated with the **World Bank API** to ensure users are viewing the most up-to-date national statistics.
* **Interactive Visualizations:** Powered by **Recharts.js**, users can view data through various chart types including:
    * Line Charts (for trend analysis)
    * Bar Charts (for categorical comparison)
    * Composed Charts (for complex data sets)
* **Comprehensive Indicators:** Explore key data points across multiple sectors:
    * **Economy:** GDP growth, Inflation rates, and Trade balances.
    * **Education:** Literacy rates and enrollment statistics.
    * **Science & Tech:** Research output and high-tech exports.
    * **Social:** Poverty headcount and demographic shifts.
* **Global Comparison:** A unique feature allowing users to compare Pakistan’s indicators against other countries to gain global context.
* **Customizable Views:** Users can toggle between different chart formats based on their preference for data consumption.
